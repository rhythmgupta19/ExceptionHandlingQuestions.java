# ExceptionHandlingQuestions.java

This repository contains a collection of Java programs that demonstrate various exception handling techniques and scenarios in Java.

## Overview

The programs cover different types of exceptions and how to handle them properly, including:

- NullPointerException handling
- Custom exception creation and handling
- Bank withdrawal scenarios with insufficient funds
- And more exception handling examples

## Files

- Question1.java through Question13.java: Individual Java programs each demonstrating a specific exception handling concept
- Questions.java: Additional question or example

## How to Run

1. Clone the repository
2. Compile with `javac <filename>.java`
3. Run with `java <classname>`

## Requirements

- Java JDK installed
- Basic understanding of Java exception handling
